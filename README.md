# 一些小发现
## 在http发送请求等待响应时,服务器对回应的处理
    1. 子线程使用epollctl 将socket添加写事件,然后主线程epoll_wait监测到写事件就绪去写回应
    2. 直接在子线程中调用write函数进行写写回应 
  虽然epoll事件成立是通过回调函数进行的O(1)返回;但是在实际中第1中方案比第2中方案要慢得多(第2中方案浏览器秒开网页,第1中要等上1秒以上)
